## Evaluation prompt

Verify each task checkbox item begins with a concrete action verb (create, write, add, update, delete, move, rename, configure, test, verify, deploy, etc.). Flag tasks starting with vague verbs: "handle", "deal with", "look into", "think about", "address", "work on", "figure out", "manage", "process".
